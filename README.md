# Zapret для macOS

Обход блокировок DPI (YouTube и другие ресурсы, заблокированные Роскомнадзором) на **macOS** — с понятным русским установщиком и графическим приложением для тех, кто не хочет возиться с терминалом.

## В основе

Проект построен на оригинальном **zapret** от bol-van, версия **v72.12**:

- Репозиторий: https://github.com/bol-van/zapret
- Релиз, взятый за основу: https://github.com/bol-van/zapret/releases/tag/v72.12 (12 марта 2025)

Сам `zapret` поддерживает macOS «из коробки» (бинарники лежат в `binaries/mac64/`, init-скрипты — в `init.d/macos/`), но готового установщика для обычных пользователей под мак не было. Этот форк добавляет именно его — вся доработка собрана в папке [`mac/`](mac/).

## Что добавлено в этом форке

Папка `mac/` — обвязка вокруг оригинального zapret специально под macOS:

- **`Zapret.app`** — графическое приложение (на AppleScript): запуск/остановка обхода, статус, редактирование списка сайтов, установка и удаление в один клик.
- **`install.sh` / `uninstall.sh`** — установка и полное удаление через терминал, с автозапуском при загрузке (launchd).
- **`start.sh` / `stop.sh` / `status.sh`** — управление обходом с корректной работой PF (системного фаервола macOS).
- **`build-app.sh`** — пересборка `Zapret.app` из исходника `ZapretControl.applescript`.
- **`config.macos`** и **`zapret-hosts.txt`** — конфиг и стартовый список сайтов (YouTube, Instagram, Facebook, X/Twitter, LinkedIn, Discord, Reddit и др.).

Также исправлены баги предыдущих сборок: зависание окна GUI на длинном выводе, неверный «зелёный» статус после остановки и конфликт запуска при активном VPN. Подробности и инструкция по установке — в [`mac/README.md`](mac/README.md).

## Как это работает

На маке zapret работает через `tpws` — небольшой прозрачный прокси. Правилами PF трафик на портах 80/443 заворачивается в `tpws`, который слегка «портит» начало TLS-рукопожатия, чтобы DPI провайдера не распознал SNI и пропустил соединение. Обход применяется **только к доменам из списка**, остальной трафик идёт напрямую. Это не VPN: IP не меняется, никакой трафик никуда не пересылается — всё происходит локально.

## Быстрый старт

```bash
cd mac
./build-app.sh        # собрать Zapret.app (без sudo)
```

Затем открыть `Zapret.app` → **🔄 Установить / Переустановить**. Полная инструкция: [`mac/README.md`](mac/README.md).

## Лицензия и авторство

Оригинальный код zapret принадлежит [bol-van](https://github.com/bol-van/zapret) и распространяется на условиях его лицензии (см. файлы проекта). Этот репозиторий — производная работа: добавлена обвязка под macOS в папке `mac/`.
