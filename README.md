# Zapret для macOS

Обход блокировок DPI (YouTube и другие ресурсы, заблокированные Роскомнадзором) на **macOS** — с понятным русским установщиком и графическим приложением для тех, кто не хочет возиться с терминалом.

## В основе

Проект построен на оригинальном **zapret** от bol-van, версия **v72.12**:

- Репозиторий: https://github.com/bol-van/zapret
- Релиз, взятый за основу: https://github.com/bol-van/zapret/releases/tag/v72.12 (12 марта 2025)

Сам `zapret` поддерживает macOS «из коробки» (бинарники лежат в `binaries/mac64/`, init-скрипты — в `init.d/macos/`), но готового установщика для обычных пользователей под мак не было. Этот форк добавляет именно его — вся доработка собрана в папке [`mac/`](mac/).

## Что добавлено в этом форке

Папка `mac/` — обвязка вокруг оригинального zapret специально под macOS:

- **`Zapret.app`** — графическое приложение (на AppleScript): **тумблер «Включить/Выключить»** (без пароля), переключатель **стратегии обхода**, статус, редактирование списка сайтов, установка и удаление в один клик.
- **`install.command`** — двойной клик для установки без терминала.
- **`install.sh` / `uninstall.sh`** — установка и полное удаление, с автозапуском при загрузке (launchd).
- **`vpn-watch.sh`** — автопауза обхода при включении VPN и автовозврат при выключении.
- **`strategy.sh`** — переключение стратегии DPI-обхода из набора проверенных пресетов.
- **`start.sh` / `stop.sh` / `status.sh`** — управление обходом с корректной работой PF (системного фаервола macOS).
- **`build-app.sh`** — пересборка `Zapret.app` из исходника `ZapretControl.applescript`.
- **`config.macos`** и **`zapret-hosts.txt`** — конфиг и стартовый список сайтов (YouTube, Instagram, Facebook, X/Twitter, LinkedIn, Discord, Reddit и др.).

Главные удобства этой сборки:

- **Тумблер без пароля** — включение/выключение в один клик (правило в `/etc/sudoers.d/zapret` на конкретные скрипты zapret).
- **Совместимость с VPN** — `vpn-watch` сам ставит обход на паузу, пока поднят VPN, и возвращает после.
- **Не забудешь, что включён** — при автозапуске после перезагрузки окно приложения открывается само.
- **Смена стратегии в GUI** — если провайдер «закрутил гайки», перебери пресеты без правки конфига.

Также исправлены баги предыдущих сборок: зависание окна GUI на длинном выводе, неверный «зелёный» статус после остановки и конфликт запуска при активном VPN. Подробности — в [`mac/README.md`](mac/README.md).

## Как это работает

На маке zapret работает через `tpws` — небольшой прозрачный прокси. Правилами PF трафик на портах 80/443 заворачивается в `tpws`, который слегка «портит» начало TLS-рукопожатия, чтобы DPI провайдера не распознал SNI и пропустил соединение. Обход применяется **только к доменам из списка**, остальной трафик идёт напрямую. Это не VPN: IP не меняется, никакой трафик никуда не пересылается — всё происходит локально.

## Установка

Нужен Mac на macOS (Apple Silicon M1/M2/M3/M4 или Intel). Есть два способа — выбери любой.

### Способ 1. Без терминала (проще)

1. Скачай проект архивом: вверху страницы зелёная кнопка **Code → Download ZIP**
   (или возьми ZIP со страницы [Releases](https://github.com/glalker/Zapret-mac-m2-git/releases)).
2. Распакуй (двойной клик по скачанному `.zip`).
3. Зайди в распакованную папку → в подпапку **`mac`**.
4. Дважды кликни по файлу **`install.command`**.
   - Если macOS напишет «не удаётся открыть, неизвестный разработчик» —
     кликни по `install.command` **правой кнопкой → Открыть → Открыть**.
5. Откроется окно `Zapret.app` → нажми **🔄 Установить / Переустановить**,
   введи пароль администратора один раз. Готово.

### Способ 2. Одной командой в терминале

Скопируй и вставь целиком (Terminal.app), нажми Enter:

```bash
git clone https://github.com/glalker/Zapret-mac-m2-git.git && cd Zapret-mac-m2-git/mac && ./build-app.sh && open Zapret.app
```

Затем в открывшемся окне нажми **🔄 Установить / Переустановить** и введи пароль.

---

После установки управляй обходом через окно `Zapret.app`:
**⏻ Включить / Выключить** (уже без пароля), **🧪 Стратегия обхода**,
**📊 Статус**, **📝 Список сайтов**. Полная инструкция и решение проблем —
в [`mac/README.md`](mac/README.md).

## Лицензия и авторство

Оригинальный код zapret принадлежит [bol-van](https://github.com/bol-van/zapret) и распространяется на условиях его лицензии (см. файлы проекта). Этот репозиторий — производная работа: добавлена обвязка под macOS в папке `mac/`.
